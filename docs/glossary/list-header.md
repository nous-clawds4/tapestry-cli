# List Header

Any node that has [List Items](./list-item.md) (elements). A List Header becomes a [Concept Header](./concept-header.md) when it is also the nodeFrom of a [class thread initiation](./class-thread-initiation.md) (IS_THE_CONCEPT_FOR) relationship.

In the nostr context, a kind 39998 event.
