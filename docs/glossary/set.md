# Set

A subset of elements within a concept (e.g., "London coffee houses" within the coffee house concept). Connected to the [Superset](./superset.md) or another Set via IS_A_SUPERSET_OF ([class thread propagation](./class-thread-propagation.md)). Connected to [Elements](./element.md) via HAS_ELEMENT ([class thread termination](./class-thread-termination.md)).

Word types: `["word", "set"]`
