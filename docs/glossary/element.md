# Element

An individual instance of a concept (e.g., a specific coffee house). The terminal node of a [class thread](./class-thread.md). An element's membership in a concept can be declared explicitly (via a HAS_ELEMENT relationship) or implicitly (via a z-tag pointing to the [Concept Header](./concept-header.md)'s [uuid](./uuid.md)).

The element's [word type](./word-type.md) is the concept's own type (e.g., `coffeeHouse`), not a generic "element" type.
