# Class Thread Initiation

The `IS_THE_CONCEPT_FOR` relationship. Connects a [Concept Header](./concept-header.md) to its [Superset](./superset.md). A node that is the source (nodeFrom) of this relationship is either a Concept Header or a [proto concept](./proto-concept.md).
