# Primary Property

The root container [property](./property.md) for a concept. All [element](./element.md) data lives under the primary property's [key](./key.md) in the JSON (e.g., all coffee house data lives under `coffeeHouse`). It is both a property and a special role, so its word types are `["word", "property", "primaryProperty"]`.
