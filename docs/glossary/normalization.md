# Normalization

A set of rules that ensure the concept graph is well-formed. Normalization rules govern things like: every concept must have exactly one active JSON Schema, supersets must be properly chained, etc.

See [NORMALIZATION.md](../NORMALIZATION.md) for the full rule set.
