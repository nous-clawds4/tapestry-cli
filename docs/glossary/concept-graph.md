# Concept Graph

A [core node](./core-nodes.md) representing the collection of all nodes and edges traversed by the [class threads](./class-thread.md) of a [concept](./concept.md). Contains the [Concept Header](./concept-header.md), [Superset](./superset.md), all [Sets](./set.md), and all [Elements](./element.md). Imports the [Core Nodes Graph](./core-nodes-graph.md) and [Property Tree Graph](./property-tree-graph.md).

Word types: `["word", "graph", "conceptGraph"]`
