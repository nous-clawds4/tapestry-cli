# Superset

The universal set for a concept — "the superset of all X." Connected to the [Concept Header](./concept-header.md) via IS_THE_CONCEPT_FOR ([class thread initiation](./class-thread-initiation.md)). All [Sets](./set.md) and [Elements](./element.md) of the concept are reachable from the Superset via propagation and termination edges.

The set of all supersets is a subset of the set of all sets — every Superset is also a [Set](./set.md). It is one of the 8 [core nodes](./core-nodes.md) for a concept, and uses plural naming (e.g., "coffee houses" not "coffee house").

Word types: `["word", "set", "superset"]`
