# Proto Concept

A node that is the source of a [class thread initiation](./class-thread-initiation.md) relationship (IS_THE_CONCEPT_FOR) but has no [elements](./element.md) — whether defined explicitly or implicitly. It has the *structure* of a concept but no *instances* yet.
