# Class Thread Termination

The `HAS_ELEMENT` relationship. Connects a [Set](./set.md) (or [Superset](./superset.md)) to an [Element](./element.md). Can be defined explicitly (as a relationship in the graph) or *implicitly* inferred from the presence of a z-tag on the element pointing to the [Concept Header](./concept-header.md)'s [uuid](./uuid.md).
