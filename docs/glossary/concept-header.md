# Concept Header

Also known as the *class thread header node*. A [List Header](./list-header.md) that is the nodeFrom of a [class thread initiation](./class-thread-initiation.md) (IS_THE_CONCEPT_FOR) relationship. This is what distinguishes a Concept Header from a mere List Header — the presence of the IS_THE_CONCEPT_FOR edge elevates it.

Any given [concept](./concept.md) can be identified by its Concept Header node.
