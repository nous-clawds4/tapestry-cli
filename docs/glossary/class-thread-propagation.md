# Class Thread Propagation

The `IS_A_SUPERSET_OF` relationship. Connects a [Superset](./superset.md) to a [Set](./set.md), or a Set to a smaller Set. Zero or more propagation steps may occur in a single [class thread](./class-thread.md).
